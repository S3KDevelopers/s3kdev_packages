## Mobile Framework Demo

### Step 1
 Go to Framework 7 folder

### Step 2
  Type `http-serve -a localhost -p 8100 -c-1` to host the server.
```
 Starting up http-serve for ./
 Available on:
   http://localhost:8100
 Hit CTRL-C to stop the server
```
### Step 3
 open `localhost:8100` to view the demo
